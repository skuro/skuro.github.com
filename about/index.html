About me

November 20, 2011

Patrias est mellus a ddas tenni prus de una

Joice Lussu

This little piece of blogosphere is owned by skuro, a.k.a. Carlo Sciolla. I'm an IT professional by day and Clojurian, meetup organizer, open source developer, biker and, well, blogger by night.

Work stuff

I grew up from fiddling with embedded linux systems, to mobile development, to mainstream Java™ backend development. I got certified as an Alfresco Recognized Developer on 2011.

I currently work at Backbase, where I act as the Technical Managerfor the server components of the Backbase CXP product.

Clojure

This awesome functional programming language is the current sink of most of my spare time. Besides coding, I try to contribute to its already great community by organizing the Amsterdam Clojurians Meetup.

Open Source

There simply isn't any other way of doing software (said the guy that works for a closed source software company). I contribute at different levels to the following open source projects:

  • plan-maven-plugin -- Unravel your maven build
  • lambdalf -- Clojure bridge for Alfresco
  • spring-webscripts-clojure-plugin -- The name tells it all
  • maven-alfresco-archetypes -- Properly build your Alfresco projects
